🏆 4-Wochen Battle-System – Regelübersicht

1️⃣ Teamwertung (Cycle-Wertung)

Die Teams sammeln gemeinsam Punkte über 4 Wochen.

🔹 Anwesenheit

  • ✅ 2 Punkte pro Spieler bei Trainingsanwesenheit
  • Verletzte Spieler erhalten die Punkte, wenn sie erscheinen
  • Krank gemeldete Spieler mit Attest erhalten die Punkte
  • ❌ Unentschuldigtes Fehlen = 0 Punkte

Die Punkte zählen zum Teamkonto.

🔹 Abschlussspiel (Training)

EreignisPunkte
Sieg+5 Punkte
Comeback-Sieg (Rückstand zur Halbzeit)+2 Bonus
Clean Sheet+2 Bonus

🔹 Team-Wettkämpfe im Training

ErgebnisPunkte
Sieg+2 Punkte
Unentschieden+1 Punkt je Team

Zusätzlich möglich:

  • 🔥 Offensiv-Challenge erfüllt → Sonderpunkte
  • 🧱 Defensiv-Vorgabe erfüllt → Sonderpunkte
  • 🎯 Trainer-Task erfüllt → Sonderpunkte

🔹 Dienstags: Challenge-Modus

Spieler können:

  • 1vs1
  • 2vs2
  • 3vs3

Gewinner erhält:

  • 🟢 +1 Punkt für sein Team
  • 🟢 +1 Challenge-Sieg in persönlicher Statistik

🔹 Private Challenges (außerhalb Training)

Möglich:

  • Tischtennis
  • Tischkicker
  • Darts
  • etc.

Gewinner erhält:

  • +1 Challenge-Sieg (persönlich)
  • ggf. +1 Team-Punkt (wenn ihr das so definiert)

👉 Auch verletzte Spieler können teilnehmen.

2️⃣ Individuelle Punkte (XP-System)

Jeder Spieler sammelt persönliche XP über die gesamte Saison.

XP kommen aus:

  • Anwesenheit
  • Team-Siegen
  • Challenges
  • Sonderaktionen
  • ggf. MVP-Wertung

XP bestimmen:

  • 📈 Level
  • 📊 Ranking
  • Hall of Fame

3️⃣ Skill-System (2-Stufen-System)

Es gibt 5 Kategorien:

  • 🧱 Defensivmonster
  • ⚽ Tormaschine
  • 🏋 Maschine
  • 🗣 Anführer
  • 🎯 Playmaker

🔹 Season Skills (0–5)

  • Werden innerhalb eines 4-Wochen-Zyklus gesammelt
  • Maximal 5 Punkte pro Kategorie
  • Werden nach Zyklusende zurückgesetzt

🔹 All-Time Skills (0–5)

  • Permanent
  • Saisonübergreifend sichtbar
  • Hall-of-Fame relevant

⭐ Promotion-Regel

Erreicht ein Spieler:

5 Season-Skill-Punkte in einer Kategorie

Dann erhält er:

  • ➡ +1 permanenten All-Time Skill Punkt in dieser Kategorie
  • ➡ Season-Skill wird im nächsten Zyklus wieder auf 0 gesetzt

4️⃣ Level-System

Alle XP erhöhen:

  • 📈 Level
  • Fortschrittsbalken
  • Ranglistenposition

Am Saisonende:

  • Erreichtes Max-Level wird gelockt
  • Eintrag in Hall of Fame

5️⃣ Hall of Fame

Dort werden gespeichert:

  • Höchstes erreichtes Level
  • Meiste XP
  • Meiste All-Time Skill Punkte
  • Meiste Challenge Siege
